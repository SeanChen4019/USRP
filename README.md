# USRP 无线多媒体传输演示平台

基于 MATLAB 2025a + Ettus X310 USRP 的**分块渐进式无线传输演示系统**，支持图像、视频、文本三种媒体的无线传输与恢复。

## 功能模式

| 模式 | TX_MODE | 说明 |
|------|---------|------|
| 仅图像 | `1` | 图片 8×8 分块传输，逐块点亮，缺失=黑色 |
| 仅视频 | `2` | 视频 20 帧缩略图网格传输 |
| 图像+视频 | `3` | 同时传输图片和视频 |
| 文本 | `4` | UTF-8 文本无线传输，终端打印 |

## 快速开始

1. 两端都设为相同模式（`TX_MODE` / `RX_MODE`）
2. **发射端**：`cd 发射机 && Transmitter_Main_UDPui0416`
3. **接收端**：`cd 接收机 && Receiver_main_UDPui0416`
4. 接收端自动恢复并保存到 `recovered/` 目录

详细说明见 [使用文档](USAGE.md)。

## 系统架构

```
发射端 (发射机/)                      接收端 (接收机/)
┌──────────────────────┐             ┌──────────────────────┐
│ 图片分块 / 视频帧提取  │             │ 预存块数据 (CRC校验)   │
│ 文本 UTF-8 编码       │             │ 文本缓冲              │
├──────────────────────┤             ├──────────────────────┤
│ 块元数据 → LDPC 编码  │   USRP     │ LDPC 译码 → 块元数据   │
│ QPSK/BPSK 调制       │  ╔════╗    │ QPSK/BPSK 解调        │
│ 升余弦滤波            │──║空口║──→│ 升余弦滤波             │
├──────────────────────┤  ╚════╝    ├──────────────────────┤
│ 固定 2 轮 + 空闲帧    │             │ CRC 校验 → 恢复        │
│ 反向链路 SNR 接收     │←── ─ ─ ─ ─│ 反向链路 SNR + 载频发送 │
└──────────────────────┘             └──────────────────────┘
```

## 物理层参数

| 参数 | 值 |
|------|-----|
| 调制方式 | QPSK (常规) / BPSK+SF15扩频 (抗扰) |
| 信道编码 | LDPC (648, 486)，码率 ~0.75 |
| CRC | CRC-32 (Ethernet 标准多项式) |
| 成形滤波 | 升余弦，滚降系数 0.25，4倍过采样 |
| 帧同步 | 511 位 PN 序列滑动相关 |
| 频率范围 | 2.0 ~ 4.0 GHz |
| 采样率 | 390.625 kHz (200 MHz / 512) |

## 数据包格式

每物理包 44 字节载荷，LDPC + CRC 保护：

**图像/视频包 (type=0/1):**

| 字节 | 字段 | 说明 |
|------|------|------|
| 0 | block_row | 块行索引 |
| 1 | block_col | 块列索引 |
| 2 | total_rows | 总行数 |
| 3 | total_cols | 总列数 |
| 4 | block_type | 0=图像, 1=视频 |
| 5-8 | crc32 | 块数据 CRC32 |
| 9-43 | 保留 | 零填充 |

**文本包 (type=2):**

| 字节 | 字段 | 说明 |
|------|------|------|
| 0 | pkt_idx | 包序号 |
| 1 | 保留 | 0 |
| 2 | total_pkts | 总包数 |
| 3 | 保留 | 0 |
| 4 | block_type | 2=文本 |
| 5-8 | 保留 | 0 |
| 9-43 | text_data | UTF-8 文本 (35字节/包) |

## 设计理念

- **非无限广播**：固定 2 轮发送，发完自动停止
- **优雅降级**：缺失块黑色填充，画面可辨识
- **收发匹配**：空闲帧 + 空闲超时机制
- **反向链路**：接收端回传承载频索引和 SNR
- **模式切换**：一个参数在 4 种传输模式间切换
